I explored Objective-C (mostly via Treehouse) and Python for a while, before taking some time out to learn HTML and CSS (and build this website). Thanks to App Academy, Ruby has quickly become my language of choice, with JavaScript a close second.

Once App Academy ends in November, I'll be applying for web development jobs in New York and London.
